Request a read-only API key

The pandabytes API uses a single read-only key, passed as the X-API-Key header (or ?key= query param). Requests are operator-approved by email: submit your address below, the operator gets an approval link, and the key is emailed to you once approved (it is never returned in a response body).

POST /v1/keys/request

Endpoint: POST https://api.restful.zone/v1/keys/request with JSON body {"email":"…"}. Returns 202 Accepted with a request_id.

Prefer curl?

curl -X POST "https://api.restful.zone/v1/keys/request" \
     -H "Content-Type: application/json" \
     -d '{"email":"you@company.com"}'

Once approved & emailed, save the key on the dashboard to live-refresh, or read the API docs.
